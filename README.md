# Ammar Hannun — Portfolio

My personal portfolio site. Built to showcase my projects and experience while applying for AI / ML / CS internships.

**Live:** [ammarhannun.github.io](https://ammarhannun.github.io)

## Stack

- HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- Inter and JetBrains Mono fonts via Google Fonts
- Light and dark mode (system-aware with a manual toggle)
- Fully responsive

## Project structure

```
.
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── headshot.jpg
    └── Ammar_Hannun_Resume.pdf
```

## Run locally

Open `index.html` directly in your browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

Hosted on GitHub Pages from the `main` branch.
