# Ammar Hannun — Portfolio

My personal portfolio site. Built to showcase my projects and experience.

**Live:** [ammarhannun.github.io](https://ammarhannun.github.io)

## Stack

- HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- Inter and JetBrains Mono fonts via Google Fonts
- Light and dark mode (system-aware with a manual toggle)
- Fully responsive
